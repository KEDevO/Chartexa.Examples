# BTC-USD Tactical View — Django

Generated by **Chartexa Pro** dashboard exporter.

## Quick start

```bash
pip install -r requirements.txt
python manage.py runserver 5000
```

Open **http://127.0.0.1:5000**

## What to edit

Edit **cx_dashboard/data_adapter.py** — implement
`DataAdapter.get_dashboard_payload()` with your data source.

## File layout

```
manage.py
settings.py
urls.py
cx_dashboard/
  urls.py
  views.py           ← do not edit
  data_adapter.py    ← YOUR DATA GOES HERE
  templates/
    dashboard.html   ← generated UI
requirements.txt
```
