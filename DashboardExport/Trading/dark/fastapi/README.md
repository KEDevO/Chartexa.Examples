# BTC-USD Tactical View — FastAPI

Generated by **Chartexa Pro** dashboard exporter.

## Quick start

```bash
pip install -r requirements.txt
uvicorn main:app --reload --port 5000
```

Open **http://127.0.0.1:5000**

## What to edit

Edit **data_adapter.py** only — implement `DataAdapter.get_dashboard_payload()`
with your data source.

## File layout

```
main.py            ← FastAPI server (do not edit)
data_adapter.py    ← YOUR DATA GOES HERE
requirements.txt
templates/
  dashboard.html   ← generated UI (do not edit)
```
