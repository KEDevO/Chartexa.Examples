# IoT MQTT Telemetry — Flask

Generated by **Chartexa Pro** dashboard exporter.

## Quick start

```bash
pip install -r requirements.txt
python app.py
```

Open **http://127.0.0.1:5000**

## What to edit

Edit **data_adapter.py** only — implement `DataAdapter.get_dashboard_payload()`
to return real data from your source (database, MQTT, CSV, REST API, etc.).

## File layout

```
app.py            ← Flask server (do not edit)
data_adapter.py   ← YOUR DATA GOES HERE
requirements.txt  ← dependencies
templates/
  dashboard.html  ← generated UI (do not edit)
```
